## Описание репозитория **RustWorks**
Данный репозиторий содержит проект **"Текстовый редактор им. Жмыха Ящерицы"**, реализованный на языке программирования Rust. Проект демонстрирует создание современного кроссплатформенного приложения с графическим интерфейсом с использованием фреймворка egui/eframe.

## Проект - [Текстовый редактор](https://github.com/MrFandore/RustWorks/tree/main/TekstRedactor)

**Цель:** Разработка полнофункционального текстового редактора с современным пользовательским интерфейсом и расширенными возможностями редактирования.

### Основной функционал:
- **Многодокументный интерфейс** - работа с несколькими файлами одновременно через систему вкладок
- **Расширенная система отмены/повтора** - до 50 шагов истории изменений
- **Интеллектуальный поиск и замена** - с поддержкой регистра и целых слов
- **Подробная статистика документа** - подсчет страниц, слов, символов, строк и абзацев
- **Автоматическое сохранение** - защита от потери данных
- **Настраиваемый интерфейс** - светлая и темная темы оформления
- **Горячие клавиши** - быстрое выполнение операций
- **Поддержка различных форматов** - текстовые файлы, Markdown, исходный код

### Технические особенности:
- **Язык программирования:** Rust
- **Графический фреймворк:** egui/eframe
- **Архитектура:** Модель-Представление с четким разделением ответственности
- **Кроссплатформенность:** Windows, Linux, macOS

## Требования для запуска

Для работы с проектом необходимо:
1. Установленный Rust (версии 1.70 или выше)
2. Установленный Cargo (менеджер пакетов Rust)
3. Любая ОС: Windows 10+, Linux (Ubuntu 18.04+), macOS 10.15+
4. Минимум 512 MB оперативной памяти
5. 10 MB свободного дискового пространства

### Рекомендуемые требования:
- 1 GB оперативной памяти
- Многоядерный процессор
- 50 MB свободного дискового пространства

## Как использовать

### Установка и запуск:

1. **Клонируйте репозиторий:**
```bash
git clone <ссылка-на-репозиторий>
cd text-editor
```

2. **Соберите проект:**
```bash
# Сборка в режиме разработки
cargo build

# Сборка в режиме релиза (рекомендуется)
cargo build --release
```

3. **Запустите приложение:**
```bash
# Запуск в режиме разработки
cargo run

# Запуск собранного приложения
./target/release/text-editor
```

## Разработка

Для разработки и модификации проекта:

1. Установите Rust и Cargo
2. Клонируйте репозиторий
3. Используйте `cargo run` для запуска в режиме разработки
4. Для сборки релизной версии используйте `cargo build --release`

### Сборка для различных платформ:
```bash
# Windows
cargo build --release --target x86_64-pc-windows-msvc

# Linux
cargo build --release --target x86_64-unknown-linux-gnu

# macOS
cargo build --release --target x86_64-apple-darwin
```
